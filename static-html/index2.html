Popis HTML

HTML je zkratka pro Hyper text markup language. Je to standardizovaný jazky pro tvorbu webových stránek, ze kterého se později vyvinulo XML. HTML může být doplněno o CSS (Cascading style sheets), pro úpravu vzhledu stránky, a javascript, pro rozšíření HTML o dynamickou část.

Příklady HTML tagů

  • H1, H2, H3, H4, H5, H6 - nadpisy
  • P - paragraf
  • Table - tabulka
  • UL, OL - Řazený seznam a neřazený seznam
  • A - odkaz
  • IMG - obrázek
  • Input - pole pro uživatelův vstup
